汪怡（1878年2月3日－1960年），浙江杭州人，別名怡安，字一庵，著名語言學家。

![汪怡](/assets/imgs/wangyi-1.jpg "汪怡"){: width="280" } ![汪怡](/assets/imgs/wangyi-4.jpg "汪怡"){: width="280" }

汪怡出生於湖南南潯鎮，其父為讀書人。10歲時，他隨父到上海經商，又輾轉到了武昌。後來，其父在武昌兩湖書院任職，汪怡也隨之同住，獲得了飽覽經書的機會。他的族親汪康年（維新派人士、著名報人）當時也在兩湖書院教書。在汪康年的鼓勵下，汪怡開始學習外藝和科學，並於自強學堂學習了算學。

### 速記學研究與創新

汪怡在速記學上深有研究，自成一家。他自幼便對音韻深感興趣，並認為漢字識讀困難，古老的切音法過於迂笨。為了將發音與文字更緊密地聯繫在一起，他參考西洋速記術，發明了一種全新的**中文速記法**。他開辦了「國語速記講習所」，自任所長並執教。在長期的教學和推廣中，他不斷完善該系統，使其最終成為主流，被廣泛應用。汪怡一生在海峽兩岸都致力於推廣其速記法。

![汪怡式速記](/assets/imgs/wangyi-2.jpg "汪怡式速記"){: width="300" }

### 教育與國語運動

汪怡年輕時曾任中學教師、營口商業學校監督、《新中國報》總編和經理等。民國初年，他任職於教育部圖書審定處，並在北京師範大學等校擔任講師，講授國音。

在國語運動中，汪怡在注音字母及國語羅馬字的制定與推廣、詞典編纂等方面作出了卓越貢獻：
- **讀音統一會**：作為會員，他在第一次會議上即提交了字母提案。
- **國語統一籌備會**：1919年成立之初即為會員，參與了注音字母頒布過程的討論。他擔任教育部特派考察員，督辦奉天等地的推行工作。1928年，他出任國語統一籌備會常務委員。
- **數人會**：他是「國語羅馬字拼音研究委員會」委員，也是發明國語羅馬字的「數人會」成員之一。

### 詞典編纂與晚年

1928年，中國大辭典編纂處正式運作，汪怡擔任**國音普通詞典股主任**。抗戰期間，他代理大辭典編纂處總主任，並在艱難的環境下自籌資金，完成了八本版《國語辭典》的編纂工作。抗戰勝利後，在此基礎上修訂出版的四本版《國語辭典》成為了經久不衰的國語標準。

![參與編纂詞典的汪怡](/assets/imgs/wangyi-3.jpg "汪怡與同仁"){: width="300" }

汪怡晚年與其長子遷居台灣，過著隱居的生活。

- 《新著國語發音學》 1924年 商務印書館
- 《中華國語最新速記學》 1928年
- 《汪怡國語速記學》 1931年
- 《國音常用字匯》 1932年（錢玄同主編，汪怡參訂）
- 《汪怡式國語速記記錄選粹》 1933年

- 《詞牌新編》 1943年

### 關於汪怡生日的考證
關於汪怡的出生日期，過往資料常有誤或缺失。根據《錢玄同日記》1937年2月3日的記載：「汪老爺今日六十大慶」。當日陰曆為丙子年臘月二十二。按虛歲推算，其陰曆生日為1877年（丁丑年）臘月二十二，對照公曆即為**1878年2月3日**。
